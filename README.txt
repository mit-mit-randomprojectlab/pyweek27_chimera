=================
“Breakout Blue“
=================

PyWeek 27 Entry for theme “6” (https://pyweek.org/27/)
Team entry by "Team Chimera" (mit-mit and Lucid Design Art)

=============
Requirements:
=============

  Python 2/3 (tested on 2.7.6 and 3.6): http://www.python.org/
  Pygame 1.9+: http://www.pygame.org/

=================
Running the game:
=================

Open a terminal/console/shell, "cd" to the game directory and run the command

  python run_game.py

================
Playing the game
================

Background Story:

The legendary cat burglar crew known as the “Stealthy Six” have finally bee apprehended and locked up in Blueland State Penitentiary! Society deems that they will rots there for the rest of their lives.

The Game:

The “Stealthy Six” on the other hand have a plan to escape! Control your six players together to escape from prison: use items to distract guards, break walls, unlock doors and make your way to freedom!

Controls:

Use the arrow keys to control each player. Tab to switch between players, space to pickup/drop/throw/use items.
